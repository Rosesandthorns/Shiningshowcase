# **App Name**: Rosie's Shiny Pokémon

## Core Features:

- Pokemon List: Display a list of all Pokémon, with a visual indication of whether their shiny form has been viewed.
- Shiny Display: Display the standard and shiny sprites of a selected Pokemon.
- Daily and Random display: Display daily featured pokemon, and feature a random one as well
- Search Functionality: Search for Pokemon by name or species.
- Filtering by Tags: Filter Pokemon by tag(s) or lack of tag(s).

## Style Guidelines:

- Light mode primary color: #118bbf.
- Dark mode primary color: #0a5778.
- Use 'Inter' sans-serif font for clear readability and a modern feel.
- Employ a flexible flexbox layout for the main content, allowing for responsiveness and adaptability across different screen sizes.
- Incorporate subtle hover effects on cards to provide visual feedback and enhance interactivity.
- Card background color: rgba(255, 255, 255, 0.85) in light mode, rgba(0, 0, 0, 0.3) in dark mode.
